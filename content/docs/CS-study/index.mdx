# 为何需要自学

如果你认真阅读了前文推荐的 wiki 连接，你会发现，无一例外的，每个 wiki 中都强调了自学的重要性，
自然，这里也不能免俗 : )

国内高校中的 cs 教育明显不满足于工业界的需求，说不定你每天在教室看到的 ppt 年龄比你都大，而 cs 作为一个更新迭代速度飞快的学科, 如果没有自学，
很有可能面临着毕业即失业的未来（这可没有危言耸听...）。

但也没有那么糟，只要你大学四年花了一些时间自学，那么相信你在就业中还是会有很大的竞争力的。


> 「以下汇聚了一些不错的学习资源」


# CS 自学指南

[cs 自学指南](https://csdiy.wiki) 是由北大的 PKUFlyingPig 学长发起的一份计算机自学指南，里面汇聚了非常不错的 cs 指南，但是需要知道的是，
里面的课程内容难度颇大，需要对计算机知识有了一定储备后才能开始。

# roadmap

[roadmap](https://roadmap.sh) 是目前 github 中 star 数量最多的项目（如果我没有记错的话），你可以在这里查看你感兴趣的方向，然后跟着路线图一步一步往下走。
相较于 cs 自学指南， roadmap 更多是面向就业的路线图。
